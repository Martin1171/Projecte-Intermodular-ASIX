# 5. Monitorització

Zabbix permet visualitzar l’estat dels dispositius mitjançant diferents apartats:

## Latest Data
Mostra dades en temps real (CPU, RAM, xarxa, disc).

## Graphs
Mostra l’evolució dels recursos del sistema.

## Problems
Mostra alertes generades pels triggers.

## Dashboard
Mostra una visió global del sistema monitoritzat.
